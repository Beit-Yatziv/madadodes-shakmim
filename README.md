# הגעת ילדי גן שקמים - מערכת מעקב הגעה

אפליקציית אינטרנט אינטראקטיבית למעקב אופן ההגעה של ילדי הגן לגן, המיועדת לשימוש על מסכי מגע.

## 📋 תיאור הפרויקט

אפליקציה פשוטה ואינטואיטיבית המאפשרת לילדים לבחור את שמם ולסווג את אופן ההגעה שלהם לגן:

- הליכה
- מכונית
- אוטובוס
- אופניים

האפליקציה מציגה את מספר הילדים שסווגו בכל קטגוריית תחבורה ומאפשרת איפוס מהיר.

## 🖥️ פלטפורמת יעד

**מסך מגע:** Philips 32BDL3751T  
**דפדפן:** דפדפן מובנה במסך (כרומבוק/Chromium based)

## 🛠️ טכנולוגיות

- **HTML5** - מבנה הדף
- **CSS3** - עיצוב רספונסיבי עם Media Queries
- **Vanilla JavaScript** - לוגיקה ללא תלות בספריות
- **Font Awesome 6.5.1** - אייקונים (דרך CDN)
- **GitHub Pages** - אירוח והפצה

## ✨ תכונות

- ✅ ממשק ידידותי לילדים עם הנחיות ברורות
- ✅ תמיכה מלאה במגע (touch events)
- ✅ עיצוב רספונסיבי (תמיכה במסכים שונים)
- ✅ אייקונים צבעוניים ומובחנים לכל סוג תחבורה
- ✅ מערכת צבעים מותאמת לכל קטגוריה
- ✅ איפוס מהיר לכל הנתונים
- ✅ אנימציות והדגשות חזותיות

## 🚀 איך להריץ

### אפשרות 1: GitHub Pages (מומלץ)

1. העלה את הפרויקט ל-GitHub repository
2. הפעל GitHub Pages מ-Settings → Pages
3. בחר branch (לרוב `main` או `master`)
4. האפליקציה תהיה זמינה ב-`https://[username].github.io/[repository-name]`

### אפשרות 2: שרת מקומי

1. פתח את התיקייה בפקודה:

```bash
cd madadodes-shakmim
```

2. הפעל שרת HTTP מקומי:

```bash
# Python 3
python -m http.server 8000

# או Node.js (אם מותקן)
npx serve
```

3. פתח בדפדפן:

```
http://localhost:8000
```
